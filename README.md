# Telegram Job Digest Bot

Бот для сбора вакансий из Telegram-каналов, анализа релевантности и отправки персонализированного дайджеста.

## Возможности MVP

- ✅ Загрузка резюме (файлом или текстом)
- ✅ Добавление списка Telegram-каналов
- ✅ Ежедневный сбор новых постов
- ✅ Выделение вакансий из постов
- ✅ Deduplication (защита от дубликатов)
- ✅ Scoring релевантности на основе резюме
- ✅ Ежедневный дайджест
- ✅ Ручной запуск дайджеста
- ✅ Настройка фильтров

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте переменные окружения:
```bash
export TELEGRAM_BOT_TOKEN="your_bot_token"
export ADMIN_USER_ID="your_telegram_id"
export DATABASE_PATH="data/vacancies.db"
export DAILY_DIGEST_TIME="09:00"
export MIN_RELEVANCE_SCORE="0.3"
```

3. Запустите бота:
```bash
python bot/bot.py
```

## Выходные данные по каждой вакансии

Каждая вакансия в дайджесте содержит:

- **Должность** (title)
- **Компания** (company)
- **Краткое описание** (description)
- **Ссылка** на оригинальный пост
- **Score** релевантности (0-100%)
- **Почему подходит** (match reasons)
- **Риски / пробелы** (risks)
- **Дата публикации**

## Как это работает

### 1. Парсинг вакансий
Бот использует паттерны для извлечения данных из постов:
- Должность (title)
- Компания (company)
- Зарплата (salary)
- Локация (location)
- Описание (description)

### 2. Scoring релевантности
Алгоритм оценивает вакансию по критериям:
- Соответствие целевым ролям (40%)
- Совпадение навыков из резюме (30%)
- Формат работы и локация (15%)
- Зарплата (15%)

### 3. Deduplication
Вакансии проверяются по уникальной паре `(channel_id, message_id)`, что исключает дубликаты.

### 4. Ежедневный дайджест
В настроенное время бот:
- Собирает новые вакансии из каналов
- Фильтрует по минимальному скору
- Формирует персонализированный дайджест
- Отправляет пользователю
- Помечает вакансии как отправленные

## Примечания

### Для полноценной работы с каналами

Для автоматического чтения сообщений из каналов потребуется дополнительно настроить Telegram Client API:

1. Получите `TG_API_ID` и `TG_API_HASH` на https://my.telegram.org
2. Установите Telethon или Pyrogram
3. Добавьте бота в администраторы каналов (опционально)

### Ограничения MVP

- Автоматический сбор из каналов требует дополнительных настроек (Telethon/Pyrogram)
- Парсинг PDF/DOC резюме упрощен (лучше работает с TXT)
- Scoring основан на ключевых словах (можно улучшить ML-моделью)
